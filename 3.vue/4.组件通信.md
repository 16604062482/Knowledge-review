- 1.props/$emit

  > 父组件 A 通过 props 的方式向子组件 B 传递，B to A 通过在 B 组件中 $emit, A 组件中 v-on 的方式实现。

  - 1.父组件向子组件传值：props

  - 2.子组件向父组件传值（通过事件形式）：$emit

- 2.$emit/$on

  > 通过一个空的 Vue 实例作为中央事件总线（事件中心），用它来触发事件和监听事件,实现任何组件间的通信，包括父子、兄弟、跨级。

  ```js
      var Event=new Vue();
      Event.$emit(事件名,数据);
      Event.$on(事件名,data => {})
  ```
